# MNIST Handwritten Digit Classification using CNN

This project uses Convolutional Neural Network (CNN) to classify handwritten digits from the MNIST dataset.

## Tech Used
- Python
- TensorFlow / Keras
- CNN (Deep Learning)

## Model Architecture
- Conv2D
- MaxPooling
- Dense Layers
- Softmax Output

## Accuracy
~98% test accuracy

## How to Run
pip install -r requirements.txt
python train.py
